快速入门

欢迎使用 Aeio Dev Agent！本指南将帮助您在几分钟内开始使用 AI 驱动的开发平台构建 Web 应用。

1. 注册账号

首先，您需要创建一个账号。访问注册页面填写所需信息：

  • 用户名：选择一个易于记忆的用户名
  • 邮箱：提供有效的电子邮件地址
  • 密码：设置一个安全的密码

立即注册 →

2. 登录系统

注册完成后，使用您的邮箱和密码登录系统：

  • 访问登录页面
  • 输入注册时使用的邮箱和密码
  • 点击"登录"按钮

立即登录 →

3. 创建第一个会话

登录后，您可以创建一个新的开发会话：

  • 点击左侧边栏的"新建会话"按钮
  • 为您的会话输入一个描述性标题（如"个人网站"、"登录页面"等）
  • 系统会自动为您创建一个隔离的沙箱环境
  • 每个会话都有独立的文件系统，互不干扰

4. 与 AI 对话生成代码

现在，您可以用自然语言与 AI 对话来生成代码：

  • 在聊天输入框中描述您想要的功能
  • 使用清晰、具体的指令会获得更好的结果

示例提示词：

  • "创建一个包含标题、介绍内容和联系按钮的个人介绍页面"
  • "做一个登录表单，包含用户名、密码输入框和登录按钮"
  • "设计一个产品展示卡片，包含图片、标题、描述和购买按钮"
  • "创建一个响应式的导航栏，包含 Logo 和几个菜单链接"

5. 预览和分享

在开发过程中，您可以随时预览和分享您的作品：

  • 实时预览：右侧预览窗口会实时显示您的代码效果
  • 独立访问：点击"独立预览"按钮可以打开全屏预览页面
  • 分享链接：设置会话为公开状态后，分享链接给他人查看
  • 导出代码：您可以将生成的代码复制到自己的项目中

下一步

掌握了基本使用方法后，您可以：

  • 浏览常见问题解答疑惑
