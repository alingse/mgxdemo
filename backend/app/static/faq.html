常见问题

以下是用户常见问题的解答，涵盖账户、使用、技术等方面。

账户相关

如何注册账号？
访问注册页面，填写用户名、邮箱和密码即可完成注册。注册后系统会自动为您创建一个访问令牌，无需邮箱验证。
忘记密码怎么办？
目前暂不支持自助密码重置功能。请通过 联系我们 页面联系管理员重置密码。
如何修改账户信息？
目前用户名和邮箱暂不支持自助修改。如需修改，请联系管理员。
如何删除账户？
删除账户会永久移除您的所有数据，包括会话、消息和沙箱文件。如需删除账户，请通过联系方式联系我们，确认后我们会为您处理。
Aeio Dev Agent 是免费的吗？
目前 Aeio Dev Agent 处于测试阶段，完全免费使用。未来可能会根据使用量提供不同的付费计划。

使用相关

如何创建第一个会话？
登录后，点击左侧边栏的"新建会话"按钮，输入会话标题即可。系统会自动为您创建一个包含默认文件（index.html、style.css、script.js）的沙箱环境。
如何让 AI 修改代码？
直接在聊天框中描述您想要的修改。使用"创建"、"修改"、"添加"、"删除"等关键词可以触发 AI 的文件修改功能。例如："把标题改成红色"或"添加一个登录按钮"。
AI 支持哪些编程语言？
目前主要支持 Web 前端开发相关的语言和格式，包括 HTML、CSS、JavaScript。您也可以使用 JSON 配置文件等文本格式。
如何下载生成的代码？
您可以通过沙箱文件管理功能查看每个文件的内容，手动复制代码到本地。未来版本将支持一键导出整个项目。
可以分享我的项目吗？
可以。每个会话都有独立的预览链接，您可以将链接分享给他人查看。预览是只读的，他人无法修改您的代码。
会话数量有限制吗？
目前没有会话数量限制，您可以创建任意数量的会话。每个会话独立运行，互不干扰。

技术相关

支持哪些浏览器？
支持所有现代浏览器，包括 Chrome、Firefox、Safari、Edge 的最新版本。建议使用 Chrome 或 Edge 以获得最佳体验。
支持移动设备吗？
支持。界面采用响应式设计，可以在手机和平板上正常使用。但推荐在桌面端进行开发，以获得更完整的编辑体验。
沙箱大小有限制吗？
每个会话的沙箱最大为 100MB，单个文件最大为 1MB。这些限制足以支持大多数 Web 项目的开发。
为什么预览不显示？
可能的原因包括：1) index.html 文件不存在或有语法错误；2) 浏览器缓存问题，尝试刷新预览；3) 代码中有不安全的操作被拦截。检查文件内容并刷新预览。
如何查看和编辑代码？
点击沙箱文件列表中的文件名即可查看内容。目前支持只读查看，如需编辑请通过 AI 对话进行修改。

AI 相关

使用的是什么 AI 模型？
我们支持多个 AI 提供商，包括 OpenAI GPT 系列和智谱 GLM 系列。系统会根据配置自动选择合适的模型。
如何写出更好的提示词？
提供具体、详细的描述会获得更好的结果。包括：1) 明确要实现的功能；2) 指定样式和布局；3) 描述交互行为；4) 提及响应式需求。参考 使用指南 中的示例。
AI 生成的代码准确吗？
AI 生成的代码通常是可靠的，但可能存在错误或不符合预期的情况。建议您检查生成的代码，并通过继续对话进行修改和优化。
我的对话内容会被用于训练 AI 吗？
您的对话内容会发送给第三方 AI 提供商进行处理。根据提供商的政策，这些数据可能会被用于改进模型。请勿在对话中分享敏感个人信息。详见 隐私政策。
AI 响应很慢怎么办？
AI 响应时间取决于多个因素，包括网络状况和 AI 提供商的服务负载。如果响应特别慢，请稍后重试或联系管理员。

高级功能

什么是 TODO 列表？
当您请求复杂功能时，AI 会自动创建任务列表来跟踪进度。每个任务完成后会自动标记为完成状态，帮助您了解开发进度。
什么是工具调用？
工具调用是 AI 使用特定功能来操作文件系统的方式，包括读取文件、写入文件、删除文件和列出文件。这使得 AI 可以直接修改您的代码。
如何使用 API？
我们提供完整的 RESTful API，支持通过编程方式管理会话、消息和沙箱文件。详见 API 文档。
支持批量操作吗？
暂不支持批量操作。每个文件操作需要单独进行。未来版本可能会添加批量管理功能。

安全和隐私

我的代码安全吗？
每个会话在独立的沙箱环境中运行，代码完全隔离。其他用户无法访问您的会话和代码。但请注意，公开分享的预览链接可被任何人访问。
数据会被备份吗？
建议定期导出您的重要代码。虽然我们会尽力保护数据安全，但无法保证在任何情况下都不发生数据丢失。
如何举报违规内容？
如发现违反服务条款的内容，请通过 联系我们 页面举报，我们会及时处理。
